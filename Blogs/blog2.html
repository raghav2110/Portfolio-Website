HTTP Status Code Definitions(Client Error 4xx)

404 Page Not Found. We see this thing popping our screen various time. But do you really know what this actually mean? In this blog we will learn what are the different HTTP status code(4xx).

The HTTP code of 4xx status appear when a webpage dosen't exist or has restricted access or rights or where client seems to have errored.These status code are applicable to any request method. There are 17 4xx status code. In this blog we will see first 5 of them. Some HTTP status code are :

  • 400 Bad Request

    • This error appear when there are some syntax error in the request made to the server.

  • 401 Unauthorized

    • This error appear when an user is denied for authorization for the credentials that are provided. The user need to provide with valid credentials for authorization.

  • 402 Payment Required

    • This code is reserved for future use.

  • 403 Forbidden

    • This error appear when server has understood the request but is refusing to fulfill it.

  • 404 Not Found

    • This error appear when server has not found any matching URL. This status code is commonly used when server is not able to reveal why request was refused.

Since we now know what are the causes of these error but, how do we fix 4xx error ? Few things can be tried and checked like :

  • Check for any spelling mistake in URL.
  • Checking for file limit or file size.
  • Clear cookies and cache.

So this was some basic about HTTP Status Code(Client Error 4xx)
